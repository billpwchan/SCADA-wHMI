
confdir2
C'est le repertoire pointé par CONFDIR utilisé pour la build en Makefile
C'est la copie exacte de celui que l'on trouve dans le produit ATS (ats_source\confdir2)

error
Repertoire generant des .h definition d'erreur à partir d'emf (generation via scadasoft)
C'est un extrait du produit ATS (contenu partiel de ats_source\server\error)

inc
Repertoire de .h necessaires provenant du produit ATS (contenu partiel de ats_source\server\inc & ats_source\server\ats\inc)

red/lib
C'est un extrait du produit ATS (contenu partiel de ats_source\server\red\lib)

scsint
C'est un extrait du produit ATS (contenu partiel de ats_source\server\ats\scsint)
Il n'y a que Trace, dont les liens avec ErrorLog ont été coupés.
On pourrait se passer de cette lib, qui reprends le code d'une fonction scada existante (ScsUserTraceFunction)
L'interet de cette lib est de pouvoir facilement stubber les traces (reprendre alors Traces_stub qui a le même code mais makefile différent)

sig/inc
C'est un extrait du produit ATS (contenu partiel de ats_source\server\ats\sig\inc)

sighilc
Les sources du fonctionnel HILC
